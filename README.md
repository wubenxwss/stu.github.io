#### 给自己的小Tips
如何修改然后重新发布呢？
在`stu.github.io`文件夹右键 Git Bash Here

```
git init 
```
上传所有文件：
```
git add .
```
或单文件:
```
git add Index.html
```
```
git commit -m "注释"
```
```
git push
```
